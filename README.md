# Nancy Stone - Author Website

A professional website for Nancy Stone, midwife, researcher, and author of "Becoming a Midwife in a Free-standing Birth Centre: I am the midwife I dreamed of becoming."

## About the Website

This website showcases Nancy Stone's work as an American-German midwife and researcher who has lived and worked in Germany for nearly forty years. The site highlights her upcoming book publication, research contributions, and academic publications.

## Features

- **Responsive Design**: Works perfectly on desktop, tablet, and mobile devices
- **Professional Layout**: Clean, elegant design appropriate for academic and professional audiences
- **Book Promotion**: Dedicated section for the upcoming book with pre-order link
- **Research Showcase**: Highlights of major research projects including the ASK a Midwife study
- **Publications**: Complete list of recent academic publications with direct links
- **Smooth Navigation**: Single-page design with smooth scrolling between sections
- **Accessibility**: Keyboard navigation support and focus states for screen readers

## Structure

- `index.html` - Main HTML structure
- `styles.css` - Complete CSS styling with responsive design
- `script.js` - Interactive JavaScript features
- `Nancy Stone 7.jpg` - Professional photo (provided)

## Sections

1. **Hero Section**: Introduction with photo and call-to-action buttons
2. **About**: Detailed biography and career statistics
3. **Book**: Information about the upcoming publication with pre-order link
4. **Research**: Key research projects and methodologies
5. **Publications**: Recent academic publications with DOI links
6. **Contact**: Professional contact information and philosophy

## Design Features

- **Color Palette**: Warm, professional colors inspired by midwifery themes
- **Typography**: Combination of Crimson Text (serif) and Inter (sans-serif) fonts
- **Animations**: Subtle scroll-triggered animations and hover effects
- **Mobile-First**: Responsive design that works on all screen sizes

## Target Audience

- Student midwives
- Newly qualified midwives  
- Midwifery educators
- Academic colleagues and researchers
- Potential book readers

## Technical Features

- Semantic HTML5 structure
- CSS Grid and Flexbox for layout

Simply open `index.html` in a web browser to view the site locally, or deploy the files to any web hosting service.
